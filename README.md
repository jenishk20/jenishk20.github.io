# jenishk20.github.io

Personal portfolio and technical blog built with Jekyll and Minimal Mistakes theme.

## Setup

```bash
# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve
```

Open http://localhost:4000

## Writing Posts

Create posts in `_posts/` with the format: `YYYY-MM-DD-title.md`

## Deploy

Push to `main` branch - GitHub Pages will automatically build and deploy.
