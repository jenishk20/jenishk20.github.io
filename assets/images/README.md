# Profile Image

Add your profile picture here as: **profile.png**

The image will appear in the left sidebar.

## Recommended Size
- 200x200 pixels or larger
- Square aspect ratio (1:1)
- PNG or JPG format

## To Add Your Image:
1. Copy your photo to this folder
2. Rename it to: `profile.png`
3. Commit and push

If you don't have an image yet, you can:
- Remove the `avatar:` line from `_config.yml` (line 29), OR
- Keep it and it will show your initials instead
